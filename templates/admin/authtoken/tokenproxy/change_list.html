API Token Guidelines

  • Each user can have only one token at a time
  • Token is automatically generated by Django (40 characters)
  • If a user already has a token, creating a new one will replace the old token
  • Tokens are used for API authentication via Authorization: Token <key> header
  • Keep tokens secure - they provide full access to the user's account
